# Shell in Go

This project is an implementation of a simple shell in Go.

## Getting Started

### Prerequisites

- Go 1.16 or higher
- gnu make installed

### Installation

Clone the repository:

```sh
git clone https://github.com/Sounak10/miniShell.git
cd miniShell
```

Build the project:

```sh
make
```

### Usage

Run the shell:

```sh
make run
```
